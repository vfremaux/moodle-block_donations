The donations block summarizes donation campains setup in Moodle with the donation module.

You can choose the scope of the agregation : single donation, all donations in the course, 
all donations in moodle.

Donations block displays a small bar giving the obtained funding ratio

Dependancies : 
==================

To be used with the DonationSingle module.

Installation : 
==================

Unzip the archive in the 'blocks' directory and run the notifications.
